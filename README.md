# Ice Cream Emporium

I cleaned it up and improved it where I could.

This is a Java project which aims to create a simple management system for an ice cream emporium and create a GUI which allows a user to use the management system. For more details, look at pdf files in assignment_instructions folder.

**This project was assigned to me during my freshman year of college by Mr. George F. Rice at UTA in his Object Oriented Class (cse1325). I originally started this project in August 2022 and finished it around November 2022.**
